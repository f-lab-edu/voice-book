# 말로쓴책

> **부모님의 이야기를 말로 기록해, 인생 한 권의 책으로 남기는 음성 기반 서비스**  
> 매일 10분 통화하듯 이야기하면, 부모님의 인생이 한 권의 책이 됩니다.

---

## 1. 프로젝트 소개

**말로쓴책**은 타이핑 없이 **오직 목소리(Voice)**로 부모님의 삶을 기록하는 서비스입니다.  
대화는 하루에 10분이면 충분합니다.  
말로 남긴 이야기는 텍스트로 정리되어, 시간이 지나 **한 권의 책(PDF/전자책/실물책)**으로 완성됩니다.

또한 목소리와 텍스트를 함께 보존해, 가족이 언제든 다시 꺼내 읽고 들을 수 있도록 합니다.

---

## 2. 핵심 가치

- **쉬움 (Zero UI/UX)**  
  복잡한 입력 없이, 말하는 것만으로 기록이 됩니다.

- **연결 (Connection)**  
  부모님의 과거와 자녀의 현재를 이어주는 소통의 통로가 됩니다.

- **영원 (Legacy)**  
  이야기뿐 아니라 **목소리 자체**를 함께 보존해, 가족의 기억으로 남깁니다.

---

## 3. 주요 기능

### 3.1 AI 라이프 인터뷰어 (Core)
정해진 질문 리스트가 아니라, 답변의 맥락을 이해하고 **꼬리에 꼬리를 무는 질문**을 이어갑니다.

예시:
- AI: “어릴 적 고향의 가을은 어땠나요?”
- 사용자: “감나무가 참 많았지…”
- AI: “가을마다 감을 따셨겠네요. 그때 누구와 같이 따셨어요?”

> 목표는 “인터뷰”가 아니라, **부담 없는 대화**로 기억을 자연스럽게 꺼내는 것입니다.

---

### 3.2 대화 기록 검토 (신뢰를 만드는 UX)
음성 → 텍스트 변환 후, 사용자가 다음 중 하나를 선택할 수 있습니다.
- 다시 녹음
- 텍스트 직접 수정
- 그대로 통과(확정)

확정된 내용만 다음 질문 생성에 사용합니다.

---

### 3.3 목소리 & 텍스트 동시 저장
- 원본 **음성(목소리)** 보관
- 검토된 **텍스트(기록)** 보관  
두 데이터가 함께 남아, “읽을 수 있고 들을 수 있는 기록”이 됩니다.

---

### 3.4 추억 소환관 (멀티미디어 매핑) *(추가 기능)*
옛날 사진(앨범)을 촬영/업로드하면, 사진 내용을 바탕으로 질문을 제안합니다.

예시:
- (결혼식 사진 업로드)
- AI: “이건 결혼식 때인가요? 이때 기분이 어떠셨어요?”

---

### 3.5 자동 편집 & 출판 (Output) *(추가 기능)*
흩어진 대화 조각을 AI가 자연스럽게 엮어 **자서전 형태**자서로 편집합니다.
- 연대기 순(유년기 → 청년기 → 중년기)
- 주제별(사랑, 고생, 교훈, 가족)

결과물:
- PDF 전자책 자동 생성
- 실물 책 주문 연동(선택)

---

### 3.6 목소리 복원 & 낭독 *(Killer Feature / 추가 기능)*
인터뷰 과정에서 수집된 음성 데이터를 기반으로 **부모님의 목소리 모델**을 생성하고,  
완성된 자서전을 **부모님의 목소리로 읽어주는 오디오북** 기능을 제공합니다.

> 서비스의 본질은 “복원 기술”이 아니라, **가족에게 부모님의 목소리를 들을 기회를 제공하는 것**입니다.

---

### 3.7 가족 연결 & 질문 요청 *(추가 기능)*
부모와 자녀를 친구 추가처럼 연결해, 자녀가 부모에게 “오늘 해보고 싶은 질문”을 보낼 수 있습니다.  
부모는 부담 없는 타이밍에 대화로 답할 수 있습니다.

---

## 4. 사용자 흐름 (User Flow)

1) AI 질문  
2) 사용자 답변(음성)  
3) 음성 → 텍스트 변환  
4) 사용자 검토: 다시 녹음 / 텍스트 수정 / 통과  
5) 확정된 답변을 바탕으로 다음 질문 생성  
6) 사용자가 질문 진행 여부 선택  
7) 대화 전체 보기 및 저장(음성+텍스트)  
8) 누적된 기록을 바탕으로 “한 권의 책”으로 편집/출력
